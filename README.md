# 介绍

根据项目 [ZH-Font-Replacement](https://github.com/ReRokutosei/ZH-Font-Replacement) 将更纱黑体的信息改为微软雅黑和宋体，用来替换 Windows 系统的默认字体。

根据[这篇帖子](https://bbs.pcbeta.com/forum.php?mod=viewthread&tid=1960120&extra=page%3D1&page=1)补充了：

1. SegUIVar.ttf
2. tahoma.ttf
3. tahomabd.ttf

根据 [yahei-sarasa](https://github.com/chenh96/yahei-sarasa) 补充了：

1. simsun.ttc
2. simsunb.ttf
